Chrome Incognito Mode Cookie Checker

This page checks to see if Chrome's Incognito Mode can read cookies that are set in non-incognito windows. It appears that it can't read cookies between browser modes.

Follow these steps to use this script:

  1. JavaScript must be enabled.
  2. Close all of your incognito windows in Chrome.
  3. Open this page in a regular (not incognito) Chrome window.
  4. Fill out the form below and click "Save".
  5. Leave this page with the submitted form open while opening a Chrome window in Incognito Mode.
  6. Visit the page in the incognito window. If you see any cookies listed at the top, then the incognito window is reading cookies from your non-incognito window.

Note: we don't save any of your data or use the cookies for anything. The source code is available here.
