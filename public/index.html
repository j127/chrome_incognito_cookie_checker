Is Chrome's Incognito Mode Really Private?

This page checks to see if Chrome's Incognito Mode can read cookies that are set in non-incognito windows.

Follow these steps to use this script:

  1. Close all of your incognito windows in Chrome.
  2. Open this page in a regular (not incognito) Chrome window.
  3. Fill out the form below and click "Save".
  4. Leave this page with the submitted form open while opening a Chrome window in Incognito Mode.
  5. Visit the page in the incognito window and click the "Get Saved Data" button.
  6. If it loads your form submission, then the incognito window and normal window are sharing cookies.
